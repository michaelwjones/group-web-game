# Blind Artists Assets

## Location

`packages/client/static/blind-artists/`

## Naming Convention

```
{zone}_{element}_{slot}.png
```

| Part | Values |
|------|--------|
| zone | `back`, `mid`, `fore`, `focus` |
| slot | `primary`, `secondary`, `highlight`, `shadow` |

Element short names:

| Element | Short name |
|---------|------------|
| Aurora | `aurora` |
| Sunset | `sunset` |
| Storm Front | `storm` |
| Mountains | `mountains` |
| Forest | `forest` |
| Lake | `lake` |
| Meadow | `meadow` |
| Rocky Shore | `rockyshore` |
| Snow Field | `snowfield` |
| Lone Tree | `tree` |
| Windmill | `windmill` |
| Campfire | `campfire` |

Examples: `back_aurora_primary.png`, `focus_tree_shadow.png`, `fore_rockyshore_highlight.png`
